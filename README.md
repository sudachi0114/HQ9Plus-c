# HQ9+.c

## 実行方法

**コンパイル・実行**

```shell
$ gcc main.c -o main
$ ./main
```

**ソースコードを読み込む**

* ファイルで読み込む場合

```shell
$ ./main hq9plus.hq9
```

* 標準入力から読み込む場合

```shell
$ echo "HQQ" | ./main
```
